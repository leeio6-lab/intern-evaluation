✅ 서버 연동 완료 — 모든 기기의 평가가 Google 스프레드시트에 중앙 저장됩니다
⚙ 서버 연동 설정 (Google Sheets 중앙 수집)
아래 4단계를 따라하면 모든 평가자의 응답이 하나의 Google 스프레드시트에 자동 저장됩니다.
한 번만 설정하면 됩니다. (약 3분 소요)
Step 1. Google 스프레드시트 생성
Google Drive에서 새 스프레드시트를 만드세요.
Step 2. Apps Script에 코드 붙여넣기
스프레드시트 메뉴에서 [확장 프로그램] → [Apps Script] 클릭 → 기존 코드를 지우고 아래 코드를 붙여넣으세요.

                    
                

            


            

                
Step 3. 웹 앱으로 배포

                

                    1. Apps Script 편집기에서 [배포] → [새 배포] 클릭

                    2. 유형 선택: "웹 앱"

                    3. 실행 주체: "나" / 액세스 권한: "모든 사용자"

                    4. [배포] 클릭 → 생성된 URL 복사
