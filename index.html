💡 사용 방법

  1. 이 HTML 파일을 웹 서버에 호스팅합니다 (GitHub Pages, Netlify, Vercel 등 무료 가능)
  2. 발표 종료 후 이 QR 코드를 화면에 띄워 청중이 스캔하도록 안내합니다

  4. "결과 대시보드" 탭에서 실시간 결과를 확인하고 엑셀로 다운로드합니다
